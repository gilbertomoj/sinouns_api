# sinouns_api


```
# Criando o Virtual env (opcional)
$ python -m venv venv

# Ativando o virtual env 
$ source venv/Scripts/Activate 


# Instalando as dependencias
$ pip install -r requirements.txt

# Iniciando o servidor
$ uvicorn main:app --host 127.0.0.1 --port 8000

```
